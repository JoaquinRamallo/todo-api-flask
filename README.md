# To-Do API (Flask)

Simple REST API built with Python and Flask to manage tasks.

## Features
- Create tasks
- List tasks
- Toggle task completion
- Delete tasks

## Tech Stack
- Python
- Flask
- REST API
- JSON

## Endpoints

GET /tasks  
POST /tasks  
PATCH /tasks/{id}  
DELETE /tasks/{id}

## Run locally

pip install -r requirements.txt
python app.py
